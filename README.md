## 🚀 Featured Projects

### 🎯 **Job Hunter** — AWS Serverless Job Automation *(Personal Project)*
> Fully automated job hunting system built on AWS Free Tier — scrapes Laravel/PHP/Node.js jobs every 6 hours, scores them against my resume using **AWS Bedrock (Claude AI)**, and delivers a smart email digest of top matches.

![AWS Lambda](https://img.shields.io/badge/AWS_Lambda-FF9900?style=flat&logo=awslambda&logoColor=white)
![DynamoDB](https://img.shields.io/badge/DynamoDB-4053D6?style=flat&logo=amazondynamodb&logoColor=white)
![Bedrock](https://img.shields.io/badge/AWS_Bedrock_AI-232F3E?style=flat&logo=amazonaws&logoColor=white)
![Node.js](https://img.shields.io/badge/Node.js-339933?style=flat&logo=nodedotjs&logoColor=white)
![SES](https://img.shields.io/badge/AWS_SES-232F3E?style=flat&logo=amazonaws&logoColor=white)

- **Architecture:** 5 decoupled Lambda functions (scraper → matcher → notifier) orchestrated via EventBridge
- **AI Matching:** AWS Bedrock Claude Haiku scores each job 0–100 against resume with keyword fallback
- **Sources:** Adzuna API + JSearch (covers LinkedIn, Indeed, Glassdoor, ZipRecruiter)
- **Cost:** ~$0/month on AWS Free Tier
- **Result:** Automated digest of only relevant jobs delivered to inbox — no manual searching

🔗 [View Repository](https://github.com/aashirhaq/job-hunter)
